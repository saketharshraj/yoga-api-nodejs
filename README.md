
# Yoga Form API

A REST API for a yoga form.




## Prerequisites

- MySql
- npm
- yarn


## Run Locally

Clone the project

```bash
  git clone https://github.com/saketharshraj/yoga-api-nodejs.git
```

Go to the project directory

```bash
  cd yoga-api-nodejs
```

Install dependencies

```bash
  yarn
```

Database migration

```bash
  cd src
  npx prisma migrate dev --name init
```

Start Server

```bash
  yarn start
```
